2. 理解与调整配置： 导入后，您会看到包含代理节点、代理策略组和规则列表的配置。 * 代理节点：列出所有可用的服务器。您可以手动测试每个节点的延迟，或根据地理位置选择。 * 策略组（如 PROXY, DOMESTIC, GLOBAL）：这是实现智能分流的关键。您可以将节点分组，并为不同规则指定使用的策略组。例如，创建一个“自动选择”组，Clash会从中挑选延迟最低的节点。 * 规则：定义了流量如何被分配。规则自上而下匹配，例如： DOMAIN-SUFFIX,google.com,PROXY 表示所有谷歌域名流量走代理。 GEOIP,CN,DIRECT 表示所有中国IP的流量直连。 MATCH,PROXY 是最终规则，表示未匹配上述规则的流量默认走代理。

3. 启动服务： 配置完成后，返回主界面，点击“启动”或“开关”按钮。系统会请求建立VPN连接的权限，务必点击“允许”或“确定”。此时，状态栏会出现一个钥匙或VPN图标，表示Clash正在运行。

4. 日志查看： 遇到连接问题时，查看客户端的连接日志（Connections 或 Logs）是极佳的排错手段，可以清晰看到每个连接被哪个规则处理、指向哪个节点。

FAQ

Mellow TUN 模式与 SOCKS5 代理有什么区别？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响指定应用。TUN 更适合全局加速，如游戏、视频和浏览器流量，提高网络稳定性和覆盖范围。
Mihomo 节点分组功能有什么作用？
节点可按地区、延迟或用途分类，结合策略组自动分流流量。便于管理大量节点，提高访问速度和连接稳定性，同时优化维护效率，方便用户快速选择最优节点。
Sing-box 如何按 IP 段实现智能分流？
Sing-box 支持根据 IP 段匹配流量，将国内流量直连、国外流量走代理。结合策略组和规则文件，实现精细化流量管理，保证访问速度和稳定性，同时提高节点使用效率。
PassWall2 的延迟测试如何影响策略组选择？
PassWall2 会根据节点延迟结果更新策略组的优先节点，自动选择延迟最低的节点处理流量。这样可以优化访问速度，提高网页浏览、视频和游戏体验的稳定性。

V2Ray gRPC 传输支持多路复用和长连接，减少握手次数。它对高延迟网络和移动网络非常友好，同时可通过 TLS 加密保护数据，提高隐蔽性和稳定性。
Potatso Lite 可以设置自定义 DNS 吗？
支持自定义 DNS，可配置本地或远程 DNS 服务器，包括 DoH/DoT。正确配置可提升解析速度、防止 DNS 污染，提高访问稳定性和安全性。
Sing-box 的配置文件与 Clash 有何区别？
Sing-box 的配置使用 JSON 格式，而 Clash 采用 YAML 格式。前者更适合程序自动解析，后者更便于人工编辑。Sing-box 在性能和扩展性上有明显优势，支持更细粒度的流量控制。
Sing-box 的自定义 outbound 功能有什么用？
Sing-box 允许用户自定义多个 outbound 出口，并通过路由规则选择性输出。例如可让流媒体走特定节点，而其他网站使用备用节点，从而实现精准分流。
VLESS + XTLS 模式能在移动网络下提高稳定性吗？
可以。XTLS 降低加密开销，减少握手次数，适合移动网络高延迟或不稳定环境。结合策略组和多出口节点，可提升连接稳定性和访问速度。
VMess AEAD 加密在高封锁网络的作用？
AEAD 加密可防止数据篡改和流量特征泄露，降低被封锁和 DPI 检测风险。保证节点长期可用，提高数据传输安全性和访问可靠性，优化用户科学上网体验。
